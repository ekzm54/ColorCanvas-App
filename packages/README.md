# Packages

ColorCanvas uses modular Swift packages.

## Modules

- ColorCanvasDomain
- ColorCanvasApplication
- ColorCanvasUI
- ColorCanvasDesignSystem
- ColorCanvasCanvasEngine
- ColorCanvasSafeColor
- ColorCanvasDocument
- ColorCanvasData
- ColorCanvasExport
- ColorCanvasResources
- ColorCanvasTesting
- ColorCanvasShared

## Dependency Rule

UI depends on Application and DesignSystem. Engines must not depend on UI.
